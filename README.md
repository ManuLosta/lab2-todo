# LAB 2 - Todo App

## How to run
1. Clone this repo
```
git clone https://github.com/ManuLosta/lab2-todo.git
```
2. Run docker compose
```
docker compose up
```
3. Go to http://localhost:5173

## Technologies used

### Backend
- NestJS
- Postgres
- JWT
- PassportJS

### Frontend
- Vite
- React
- React Hook Form
- Tanstack Query
- Tanstack Router
